# Learning roadmap for this project

## Phase 1 - SQL + MySQL

- Understand server, database, table, row, and column
- Create databases and tables
- Practice CRUD operations
- Learn joins, grouping, filtering, and sorting
- Use foreign keys and constraints
- Move into CTEs and window functions

## Phase 2 - Python ingestion

- Read CSV and JSON files
- Call APIs
- Normalize data
- Load source data into raw MySQL tables

## Phase 3 - Modeling

- Build raw, staging, and marts layers
- Define grain of fact tables
- Build dimensions and business metrics

## Phase 4 - Reliability

- Add tests for nulls, uniqueness, and referential integrity
- Make scripts rerunnable
- Add logging and basic monitoring

## Phase 5 - Orchestration and warehouse workflow

- Airflow DAGs
- dbt models and tests
